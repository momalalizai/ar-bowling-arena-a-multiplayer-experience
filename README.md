# AR Bowling Arena – Multiplayer Interactive AR Game

A Unity-based mobile augmented reality (AR) fun game that reimagines traditional bowling gameplay through a simultaneous multiplayer interaction system.
---
## Project Overview

AR Bowling Arena was designed as an experimental AR game to explore new forms of multiplayer interaction in real-world environments.

Unlike traditional bowling games where players take turns, this project introduces a **simultaneous gameplay mechanic**:

- One player plays the bowling game
- Another player interacts in real-time by placing obstacles

This creates a dynamic gameplay experience.

---
## Core Concept

The project challenges the idea of turn-based gameplay by introducing:

- Real-time interference mechanics
- Strategic or random obstacle placement
- Shared game environment between players

This transforms a simple bowling game into a **competitive and interactive AR experience**.

---
## Gameplay Mechanics

- Players place a bowling lane in a real-world environment using AR
- The main player rolls the ball toward the pins
- A second player places obstacles in real-time
- Obstacles affect ball trajectory and outcome
- Game outcome depends on both skill and interaction

---
## Features

- AR-based environment placement using mobile device
- Physics-driven bowling system
- Real-time obstacle interaction
- Swipe-based ball throwing mechanics
- Room-based UI concept for shared gameplay sessions
